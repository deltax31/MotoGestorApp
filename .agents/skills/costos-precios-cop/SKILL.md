# 💰 Skill: Costos y Precios en COP

## Objetivo
Proporcionar información precisa de costos de servicios, mano de obra, repuestos y mantenimiento de motos en pesos colombianos, ajustados por tipo de moto y región.

## Cuándo Activar Esta Skill
- El usuario pregunta "¿cuánto cuesta X?"
- El usuario quiere presupuestar un servicio
- Se necesita estimar costo de una reparación
- El usuario quiere saber si le están cobrando justo

### 3. Variación por Ciudad/Región
| Región | Factor de Ajuste | Razón |
|---|---|---|
| Bogotá | +10-15% | Mayor costo de vida |
| Medellín | Base (referencia) | Precio promedio |
| Cali | -5% a base | Similar a Medellín |
| Barranquilla/Costa | -10 a -15% | Menor costo mano de obra |
| Pueblos/rural | -15 a -25% | Pero menor disponibilidad |

### 4. Señales de Cobro Justo vs Excesivo
| Señal de Alarma | Lo que Indica |
|---|---|
| Cobra sin diagnosticar | Mala práctica, pedir evaluar primero |
| No muestra repuesto viejo | Verificar que realmente lo cambiaron |
| Precio muy por debajo del rango | Repuestos de dudosa calidad |
| Cobra "revisión de computador" en moto de carburador | Cobro innecesario |
| "Hay que abrir el motor" sin diagnosticar | Segundo diagnóstico recomendado |

### 5. Formato de Respuesta
1. **💰 Costo total estimado** — rango bajo-alto
2. **📦 Desglose** — repuestos + mano de obra por separado
3. **🏷️ Opciones** — económico / calidad media / premium
4. **📍 Variación** — ajuste por ciudad si se conoce
5. **💡 Tip** — cómo ahorrar o qué evitar
