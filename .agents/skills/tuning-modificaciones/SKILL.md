# 🔧 Skill: Tuning y Modificaciones

## Objetivo
Guiar al usuario sobre modificaciones y accesorios para su moto, priorizando la legalidad, seguridad y relación costo-beneficio en el contexto colombiano.

## Cuándo Activar Esta Skill
- El usuario quiere modificar o personalizar su moto
- El usuario pregunta sobre accesorios o mejoras
- El usuario quiere más potencia o cambiar el escape
- Se necesita evaluar si una modificación es legal

## Instrucciones para la IA

### 1. Legalidad de Modificaciones en Colombia

#### ✅ Legales (No afectan tecnomecánica)
| Modificación | Detalle |
|---|---|
| Defensas/sliders | Protección, cualquier tipo |
| Parrilla y baúl | Carga, cualquier marca |
| Protector de manos | Seguridad, recomendado |
| Luces adicionales (legales) | Que no encandilen, bien instaladas |
| Asientos aftermarket | Mismo tamaño/forma |
| Calcomanías/vinilo | Estético, sin cambiar color base en tarjeta |
| Manubrio del mismo tipo | Reemplazo compatible |
| Espejos aftermarket | Deben ser 2, funcionales |

#### ⚠️ Zona Gris (Depende de instalación)
| Modificación | Consideración |
|---|---|
| Conversión LED faro | Legal si no encandila, bien alineado |
| Direccionales LED | Deben parpadear a frecuencia correcta |
| Cambio de rines | Misma medida y especificación |
| Suspensión aftermarket | Mismas medidas, puede mejorar |

#### 🔴 Potencialmente Ilegales o Problemáticas
| Modificación | Riesgo |
|---|---|
| Escape ruidoso/deportivo | Multa por ruido + no pasa tecno |
| Aumento de cilindrada | Ilegal si no se registra en RUNT |
| Eliminación de catalizador | No pasa gases |
| Placas escondidas/modificadas | Multa grave |
| Cambio de color sin registro | Infracción documental |
| Barras LED frontales | Encandilan, multa + inmovilización |

### 4. Advertencias de Seguridad
- ⚠️ Modificar frenos solo con repuestos de calidad certificada
- ⚠️ Modificar escape puede anular garantía del concesionario
- ⚠️ Cambiar relación de transmisión afecta velocímetro
- ⚠️ Kits de aumento de cilindrada son ILEGALES si no se registran
- ⚠️ Luces estroboscópicas están prohibidas (solo vehículos de emergencia)

4. **⚡ Efecto real** — expectativa realista
5. **⚠️ Riesgos** — qué puede salir mal
