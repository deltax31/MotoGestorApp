# 🔩 Skill: Repuestos y Compatibilidad

## Objetivo
Ayudar a los usuarios a encontrar repuestos correctos, identificar compatibilidades entre modelos y marcas, y recomendar opciones por calidad y precio en el mercado colombiano.

## Cuándo Activar Esta Skill
- El usuario pregunta por repuestos para su moto
- El usuario quiere saber si un repuesto de otra moto le sirve
- El usuario busca alternativas más económicas
- El usuario necesita una referencia o número de parte

## Instrucciones para la IA

### 1. Información Necesaria para Recomendar
Siempre confirmar:
- **Modelo exacto** (marca, línea, cc, año)
- **Tipo de repuesto** (motor, frenos, eléctrico, carrocería, suspensión)
- **Presupuesto** (original, genérico calidad, económico)
- **Urgencia** (puede esperar para pedir online vs necesita hoy)

#### Filtros de Aceite (por número de parte)
- Motos pequeñas suelen usar filtro tipo cartucho
- Verificar dimensiones: largo × diámetro × rosca
- Marcas: HiFlo (HF131, HF138, etc.), K&N, Vesrah

### 4. Calidad de Repuestos — Guía de Marcas

| Nivel | Descripción | Ejemplos |
|---|---|---|
| 🏆 **Original (OEM)** | Del fabricante, mayor garantía | Genuine Yamaha, Honda, Bajaj |
| 🥇 **Premium aftermarket** | Calidad igual o superior | DID, NGK, Brembo, Showa |
| 🥈 **Buena calidad** | Relación calidad-precio | HiFlo, JT Sprockets, EBC |
| 🥉 **Económico** | Funciona pero menor durabilidad | Osaki, China genérico |
| ❌ **No recomendado** | Riesgo de seguridad | Frenos sin marca, cadenas sin certificación |

### 5. Advertencias Importantes
- ⚠️ **Frenos**: NUNCA recomendar pastillas o discos genéricos sin marca
- ⚠️ **Motor**: En piezas internas (pistón, anillos), preferir OEM o premium
- ⚠️ **Eléctrico**: Reguladores y CDI genéricos pueden dañar el sistema
- ⚠️ **Llantas**: Solo marcas reconocidas (Pirelli, Michelin, IRC, Kenda)
- ✅ **Filtros, aceite, bujías**: Buen aftermarket funciona perfecto

### 6. Formato de Respuesta
1. **✅ Repuesto exacto** — referencia OEM si se conoce
2. **🔄 Alternativa compatible** — marca reconocida, más económica
3. **💰 Rango de precio** en COP
4. **📍 Dónde conseguirlo** — Mercado Libre, almacenes de motos, distribuidores
5. **⚠️ Advertencia** si hay riesgo de incompatibilidad
6. **💡 Tip de instalación** si aplica
